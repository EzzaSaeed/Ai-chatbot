# 🤖 Neurofive Solutions AI Chatbot

An AI-powered chatbot developed as part of the **Neurofive Solutions Prompt Engineering & Generative AI Internship**.

The chatbot uses a real Large Language Model (LLM) API, a custom system prompt, and an interactive Gradio interface. It is deployed on Hugging Face Spaces.

## 🚀 Features

- Custom AI assistant persona
- Real LLM API integration
- Interactive Gradio chat interface
- Secure API key management using environment variables
- Deployed on Hugging Face Spaces
- Handles both technical and off-topic questions while staying in character

## 🛠️ Technologies Used

- Python
- Gradio
- OpenRouter API
- Hugging Face Spaces
- Prompt Engineering

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Add your OpenRouter API key as an environment variable:

```
OPENROUTER_API_KEY=your_api_key
```

4. Run the application

```bash
python app.py
```
